# 싸피 광주 5반 알고리즘 스터디

## :fire: 스터디 Rule 

공통 2문제, 자유 1문제 풀고 깃 커밋
- 매주 화요일 풀어온 문제를 한명씩 리뷰함
- 목요일은 타임어택 형식으로 1문제를 정해진 시간에 풀고, 다음주 알고리즘 선정
